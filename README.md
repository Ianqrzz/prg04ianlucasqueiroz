## 📋 Sobre o Projeto

**E-commerce 074 diversão** é uma plataforma de comércio eletrônico desenvolvida como projeto individual para a disciplina de programação web. O objetivo é criar uma loja online intuitiva e responsiva para venda de produtos de entretenimento e diversão, com uma experiência de compra agradável e segura.

---

## 📁 Estrutura de Pastas

A estrutura do projeto está organizada da seguinte forma:

```
prg04ianlucasqueiroz/
│
├── README.md                          # Documentação do projeto
├── .git/                              # Repositório Git
│
└── infraestructure/                   # Arquivos da infraestrutura web
    ├── assets/                        # Pasta de assets (estilos, scripts, mídia)
    │   ├── css/                       # Folhas de estilo
    │   │   └── global/                # Estilos globais da aplicação
    │   ├── js/                        # Scripts JavaScript
    │   ├── fonts/                     # Fontes personalizadas
    │   ├── images/                    # Imagens e ícones
    │   └── midia/                     # Arquivos de mídia (vídeos, áudios)
    └── pages/                         # Páginas HTML
        ├── index.html                 # Página principal da aplicação
        ├── atividade-3.html           # Página de atividade 3
        ├── sandbox.html               # Página sandbox para testes
        ├── about/                     # Página sobre o projeto
        └── contact/                   # Página de contato
```

### 📝 Descrição das Pastas

| Pasta | Descrição |
|-------|-----------|
| **infraestructure/** | Diretório raiz que contém todos os arquivos da aplicação web |
| **assets/** | Diretório centralizado para todos os recursos estáticos (CSS, JS, fontes, mídia) |
| **css/** | Contém todas as folhas de estilo CSS do projeto. A pasta `global/` armazena os estilos que se aplicam à toda a aplicação |
| **js/** | Armazena os arquivos JavaScript que implementam a lógica interativa e funcionalidade da aplicação |
| **fonts/** | Contém as fontes personalizadas utilizadas no design da interface |
| **images/** | Repositório para imagens e ícones utilizados no projeto |
| **midia/** | Repositório para arquivos de mídia como vídeos e áudios |
| **pages/** | Contém todas as páginas HTML do projeto, incluindo a página principal e páginas adicionais |
